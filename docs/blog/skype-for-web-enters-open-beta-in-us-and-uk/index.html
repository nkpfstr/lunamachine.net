Skype for Web Enters Open Beta in US and UK

If you’re a regular Skype user, you might want to check out the new Skype for Web beta. The set up is pretty clunky - you need to install two plugins to use it - but once you’re in, it’s a pretty familiar experience.

It’s not a full replacement for the desktop app yet (that may change when it leaves beta), but it means you can use Skype on, say, a Google Chromebook.
